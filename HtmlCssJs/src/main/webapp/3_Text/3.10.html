태그 종류에 상관없이 사용하는 글로벌 속성

class 속성

class 속성은 요소에 클래스명을 지정할 때 사용합니다. 클래스명은 CSS에서 클래스 선택자로 활용하고,같은 클래스명은 여러요소가 중복해서 가질 수 있습니다.

id 속성

id 속성은 요소에 아이디를 지정할 때 사용합니다. 아이디는 CSS에서 아이디 선택자로 활용하며, 중복될수 없습니다.

style 속성

title 속성

title 속성은 요소에 추가정보를 넣을 때 사용하는데,title 속성에 넣은 값은 요소에 마우스를 올리면 툴팁으로 표시됩니다.
